Personalización de Colores

Tailwind incluye una paleta de colores elaborada por expertos y lista para usar, perfecta para prototipos o producción.

¿Cómo funciona la escala?

Cada color tiene una escala numérica del 50 al 950.

  • 50-100: Tonos muy claros, ideales para fondos (backgrounds).
  • 400-600: Tonos medios, ideales para botones, bordes o iconos.
  • 700-950: Tonos oscuros, ideales para texto (para asegurar buen contraste).
